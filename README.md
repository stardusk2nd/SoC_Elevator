## Project Summary
### Used Tool
- Vivado | Verilog HDL
- Vitis | C

### Used Device
- FPGA Development Board: **Basys3**
- Graphic LCD (SPI-based)
- Stepper Motor
- Photo Interrupter

### Designed Custom IP
- SPI Transmission IP
- Timer and Counter IP

## Directory Descryption
1. **Common**: Verilog code commonly used across multiple IPs
2. **PWM**: PWM IP code
3. **SPI**: SPI Tx IP code
4. **vitis**: Vitis(C) code
